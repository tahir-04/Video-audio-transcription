# Universal Media Transcriber

## Overview
A web application that accepts audio files, video files, and supported URLs, extracts speech, and converts it into text using Faster-Whisper.

## Features
- Upload audio files
- Upload video files
- Process supported media URLs
- Extract audio with FFmpeg
- Generate transcripts
- Download transcripts as TXT

## Tech Stack
- React + Vite
- FastAPI
- Faster-Whisper
- FFmpeg
- yt-dlp
